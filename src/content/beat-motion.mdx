Beat Motion is a direct interface between the movement of the body and sound. It used consumer-grade hardware (Kinect) to output MIDI notes that can be used in any audio software.

This is a MIDI controller for music production built using a Kinect. It allows the user to "control knobs" using their left arm and to send notes with a 2x3 grid on the right hand.

## Role and Tools

I led a small team of four people, where I programmed the interface on C++, and produced the original soundtrack in Ableton Live.
